## Фичи

- Регистрация командных и индивидуальных участников
- Просмотр, планирование графика и регистрация для участия в соревнованиях
- Создание "своей" сборной, чтобы быть в курсе результатов
- Выбор любимых видов спорта
- Суточное он-лайн табло мероприятий и результатов соревнований
- Пуш-уведомления зарегистрированных участников
- Прямые трансляции
- Лента новостей
- Мотивирование активности населения в спортивных мероприятиях выбранного муниципалитета/региона
- Получение информации о грантах на реализацию собственных идей

## Запуск

Для запуска решения требуется Python3. 
Чтобы запустить программу, требуется установить зависимости.<br>
`pip install -r requirements.txt`

После чего можно запустить десктоп-приложение с помощью<br>
`python3 main.py`
